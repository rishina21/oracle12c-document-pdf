Description of the illustration priv_analysis_create122.png

Image priv_analysis_create122.png shows the Privilege Analysis: Create Policy page. From left to right, top to bottom, are the following components:

  • Policy field, in which you enter a name for the policy.

  • Description field, in which you can optionally enter a description for the policy.

  • Scope list, provides a choice of Database, Role, Context, and Role and Context.

To the right of these fields is a set of instructions, which are described in the surrounding text of this procedure.
