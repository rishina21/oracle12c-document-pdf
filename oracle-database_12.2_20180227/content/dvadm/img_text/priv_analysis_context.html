Description of the illustration priv_analysis_context.gif

Image priv_analysis_context shows the Policy Expression Builder. From top to bottom are the following components:

- Policy is in effect when checkbox, with a user name list, a boolean list, a second user name list, and a second boolean list.

- Policy expression field, in which you can enter an expression.

- A Validate button for the Policy Expression field.

- OK and Cancel buttons.
